# 移除页面中所有广告区域

## 1. 广告组件使用情况分析

### 广告组件文件

* `src/components/ads/AdBanner.tsx` - 广告横幅组件

* `src/components/ads/NativeAd.tsx` - 原生广告组件

### 使用广告组件的页面/组件

* `src/app/page.tsx` - 主页面，顶部和底部各1个 AdBanner

* `src/app/privacy/page.tsx` - 隐私政策页面，顶部和底部各1个 AdBanner

* `src/app/terms/page.tsx` - 服务条款页面，顶部和底部各1个 AdBanner

* `src/components/calculator/Calculator.tsx` - 计算器组件，使用了1个 NativeAd

## 2. 移除步骤

### 步骤1: 移除主页面广告

* 文件：`src/app/page.tsx`

* 移除 AdBanner 组件的导入语句

* 删除顶部和底部的 AdBanner 组件引用

### 步骤2: 移除隐私政策页面广告

* 文件：`src/app/privacy/page.tsx`

* 移除 AdBanner 组件的导入语句

* 删除顶部和底部的 AdBanner 组件引用

### 步骤3: 移除服务条款页面广告

* 文件：`src/app/terms/page.tsx`

* 移除 AdBanner 组件的导入语句

* 删除顶部和底部的 AdBanner 组件引用

### 步骤4: 移除计算器组件广告

* 文件：`src/components/calculator/Calculator.tsx`

* 移除 NativeAd 组件的导入语句

* 删除 NativeAd 组件引用

<br />

## 3. 验证

执行 `npm run build` 命令确保项目可以正常构建，没有编译错误。
